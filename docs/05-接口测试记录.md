# AI Agent 多角色圆桌对谈与剧本生成系统接口测试记录

## 一、测试说明

本文档用于记录 **AI Agent 多角色圆桌对谈与剧本生成系统** 的接口测试情况，主要验证系统后端接口是否能够正常响应，前后端联调是否稳定，以及核心 AI 应用接口是否能够完成登录鉴权、多 Agent 圆桌、Tool Calling 内容分析、RAG 知识库问答、Memory 用户记忆等功能。

本次接口测试采用 **Postman 手工测试 + Python requests 自动化测试 + pytest 回归测试** 的方式进行。

---

| 项目名称 | AI Agent 多角色圆桌对谈与剧本生成系统 |

本次接口测试主要覆盖以下接口：

| 模块 | 接口内容 |
